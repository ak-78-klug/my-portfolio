WebHawk - Blogging Website

Made a blogging website using NextJS with a fluid modern user interface. All the SEO techniques are applied to this website. Highly responsive with high-quality blog content.

Blogs are written as Markdown which is basically like HTML so that the blogs are rendered dynamically.

I make use of server-side rendering and static site generation so that the user will not read any outdated posts and stale content.

Technologies:
  • - NextJS
  • - ReactJS
  • - React Markdown
  • - React Syntax Highlighter
